# do verb trainer

A simple Korean → English practice trainer.

## Live site
https://hyidyong.github.io/-/

## How to use
- Start: begin
- Next: next question
- Show/Hide Answer: toggle answer
- Restart 30s: restart timer
- Shuffle: randomize question order
- Fix: edit the shown answer (saved on this device/browser)

## Files
- index.html: UI + logic
- questions.js: question/answer data
- corrections.json: shared answer corrections (for everyone)

## Notes
- “Fix” edits are saved locally on your device.
- To share fixes with everyone, update `corrections.json`.
